- [ ] **真浏览器（lead 亲跑亲看，本环境 deepseek 真起会话）**：
  - [ ] **AC-1 误杀根除（核心场景）**：① 跑一条用 agent X 的 run（≥1 阶段 done）→「进入完整对话」复活 X 的会话、发消息确认活；② 再起一条也用 X 的 run、跑完 X 阶段；③ 断言**用户那条会话仍活**（可继续聊）——修复前会被销毁、修复后不销毁的确定性区分判据。

  - [ ] pageErrors=0；lead 亲看截图。
- [ ] **收尾**：删 T1 探针（命门事实迁详细设计 + ADR D-R8-*）；回写 progress / 详细设计实测落地 / ADR D-R8-*（lead 实现级取舍）；registry 复位清测试产物；push（待用户授权）+ ls-remote 实测同步。
